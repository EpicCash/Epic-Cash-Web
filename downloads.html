Epic Server 2.17.0 - Updated 10-24-2021

This version accommodates the halving, and the algo change to the new 48/48/4 proportions. As of block height 1157760, mining rewards will be halved, and blocks algos will change to the following:

  • 48% ProgPow (GPU)
  • 48% RandomX (CPU)
  • 4% Cuckoo (ASIC/CPU/GPU)

Please search and replace both 2.15 files – the epic.exe and foundation.JSON with the new ones from the 2.17 downloaded folder. If you have any questions, please post in the Epic Cash Help Desk group - https://t.me/epiccashhelpdesk. For people with technical knowledge to compile their own – the source code is located https://github.com/EpicCash/epic/tree/2.17.

Please note, this is not the V3 update, that will come after the halving – stay tuned for additional instructions regarding V3.
