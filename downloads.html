Epic Server 2.11.1 - Updated 7-31-2020

This release fixes the path of foundation.json on newly generated configuration files for Windows installs and you should not have to do "epic.exe server config" on first run anymore. It also includes a compile with the latest version of the Rust compiler 1.45.

If you get a missing DLL error, VCRUNTIME140_1.dll then install this: https://support.microsoft.com/en-us/help/2977003/the-latest-supported-visual-c-downloads

Note: may require one or two reboots to take effect from reports.
